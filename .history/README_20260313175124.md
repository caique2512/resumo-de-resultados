# Resumo de Resultados

Uma página estática simples que demonstra um painel de resultados estilo "resumo de desempenho" com um cartão central exibindo uma pontuação e um resumo dos indicadores (reação, memória, verbal, visual).

## 🧩 Visão Geral

O layout apresentado na imagem mostra um componente dividido em duas partes:

- **Esquerda**: cartão de desempenho com pontuação geral e mensagem de feedback.
- **Direita**: lista de categorias (Reação, Memória, Verbal, Visual) com seus respectivos valores e um botão de ação.

> ⚡ Essa estrutura é ideal para dashboards leves, resultados de testes ou relatórios de performance.

## 🚀 Como usar

1. Clone este repositório ou faça o download dos arquivos.
2. Abra o arquivo `index.html` em um navegador moderno.

```bash
# Exemplo (Linux/macOS/Windows com WSL):
# Navegue até a pasta do projeto e abra o HTML
# No Windows você também pode dar dois cliques diretamente no index.html
start index.html
```

## 🗂 Estrutura do projeto

- `index.html` – marcação principal da página.
- `src/css/reset.css` – reset básico de estilos.
- `src/css/estilos.css` – estilos específicos para o layout de resumo.
- `src/images/` – pasta para imagens e ícones usados no projeto.

## 🛠 Tecnologias usadas

- HTML5
- CSS3 (incluindo gradientes, bordas arredondadas e layout flexbox)

## 📌 Personalização

Para alterar os valores exibidos (pontuação, categoria, cores etc.), edite diretamente o HTML em `index.html` e/ou os estilos em `src/css/estilos.css`.
